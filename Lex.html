ActionCallback(yytext, match, namespace, tokens){String|Array.<String>|*}

lib/index.js, line 403

Callback used during lexing to return tokens and/or push/pop/set state.

Name Type Description
yytext String

The full text which matched.
match Array.<String>

Any () matches or empty Array if no matches.
namespace String

The namespace for this match.
tokens Array.<String>

Any tokens there were setup for this State.
Returns:
return value which is an Array or String will be taken as tokens to be returned. Any other return is ignored.
Example
rule('myRule').action(function (yytext, match, namespace, tokens) {
  this.yytext = yytext;
  return tokens;
});
